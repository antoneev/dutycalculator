# dutycalculator

A responsive web application that estimates the full costs (including customs/internationational fees) of U.S items bought from the Bahamas.

## [About](https://antoneev.github.io/dutycalculator/about.html)

## Built with
- HTML
- CSS
- Bootstrap 4
- Javascript/JQuery
